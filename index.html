API Reference
CORS-open GitHub Pages
Method Endpoint Description
GET api/anime-ids.json Array of 1,257+ MAL IDs with count + updated timestamp
GET api/sites.json 97 streaming sites with name, url, tag, top flag
GET api/config.json Featured IDs, AnimeWeb URL, version, endpoint pointers
USAGE EXAMPLE
// Fetch anime IDs from AnimeDb
const r    = await fetch('https://ccguvycu.github.io/animedb-site/api/anime-ids.json');
const data = await r.json();
// data.ids    → [1, 19, 20, 21, ...]
// data.count  → 1257
// data.updated → "2026-05-23T..."

// Fetch streaming sites
const s     = await fetch('https://ccguvycu.github.io/animedb-site/api/sites.json');
const sites = await s.json();
// sites.sites → [{ name, url, tag, top? }, ...]
